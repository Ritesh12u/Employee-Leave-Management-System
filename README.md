## 📌 Overview

Employee Leave Management System is a Spring Boot-based web application that automates leave requests. Employees can apply, track status, and view balances, while admins manage approvals—ensuring efficiency and transparency.

---

## ✨ Features

✔️ Apply for Leave
✔️ Track Leave Status
✔️ View Leave History
✔️ Leave Balance Management
✔️ Admin Approval/Rejection
✔️ Role-Based Access Control
✔️ RESTful API Architecture

## 📡 API Endpoints

| Method | Endpoint            | Description        |
| ------ | ------------------- | ------------------ |
| POST   | /leave/apply        | Apply for leave    |
| GET    | /leave/history      | View leave history |
| GET    | /leave/status/{id}  | Check leave status |
| PUT    | /leave/approve/{id} | Approve leave      |
| PUT    | /leave/reject/{id}  | Reject leave       |
